# 塚本隆大

東京芸術大学美術学部芸術学科を卒業後、東京大学大学院総合文化研究科 超域文化科学専攻（文化人類学）修士課程へ進学。現在は、東京大学大学院総合文化研究科 超域文化科学専攻（文化人類学） 博士課程に在籍。東京大学大学院卓越大学院グローバル・スタディーズ・イニシアティブ 卓越リサーチアシスタント。
